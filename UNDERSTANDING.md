## ğŸ¯ Project Overview

The Economic Dashboard is a web application that:
- **Pulls economic data** from the BLS (Bureau of Labor Statistics) and FRED (Federal Reserve Economic Data) APIs
- **Visualizes data** on interactive choropleth maps of the United States
- **Displays state-level statistics** including unemployment, GDP, population, income, cost of living, and economic growth
- **Provides time-series animation** to see how data changes over years (1976-2024)

## ğŸ”§ Key Components

### Frontend (`static/main.js`)

**Core Features**:
1. **Choropleth Map** - Visualizes data for all 50 states + DC using Plotly.js
2. **Time Slider** - Select years from 1976-2024
3. **Animation System** - Auto-play through years with play/pause
4. **Dataset Switcher** - Six different economic metrics
5. **Leaderboard** - Dynamic state rankings

**Data Objects**:
```javascript
datasets = {
  gdp: [4200, 2600, ...],           // 51 values for each state
  population: [39.0, 30.5, ...],
  unemployment: [4.7, 4.2, ...],
  income: [84, 82, ...],
  costOfLiving: [150, 145, ...],
  growth: [70, 65, ...]
}
```

**Color Themes**: Each dataset has a custom gradient:
- GDP: Blue tones
- Population: Green tones
- Unemployment: Red/Orange/Yellow
- Income: Purple/Yellow
- Cost of Living: Teal tones
- Growth: Blue/Navy

---

### Backend Data Pullers

**BLS Data (`datapull-BLS.py`)**:
- Uses state-specific LAUS series IDs
- Supports 4 metrics: unemployment rate, employment, unemployment (raw), labor force
- Returns data as `{year: value}` dictionary

Example usage:
```python
results = get_state_data("California", 2015, 2024, "unemployment_rate")
# Returns: {'2015': 6.2, '2016': 5.4, ...}
```

**FRED Data (`datapull-FRED.py`)**:
- Framework for fetching GDP data
- Requires API key
- Currently a work in progress

## ğŸš€ How to Run

### On Linux/Mac:
```bash
chmod +x run.sh
./run.sh
```

### On Windows:
```cmd
run.bat
```

### Manual Setup:
```bash
# Create virtual environment
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# OR
.venv\Scripts\activate     # Windows

# Install dependencies
pip install fastapi[standard] requests numpy pandas matplotlib

# Run the server
fastapi dev main.py
```

Then open http://127.0.0.1:8000/home in your browser.
